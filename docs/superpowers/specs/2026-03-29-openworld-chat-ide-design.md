## 项目信息

- **项目名称**: OpenWorld Chat IDE
- **阶段**: UI 模拟（不接 AI）
- **技术栈**: Nuxt 4 + WebContainer + pnpm monorepo

## 组件设计

### AppHeader
- Logo 和项目名称
- 模式切换按钮（聊天模式 / 代码模式）
- 预览按钮

### ChatPanel
- 消息列表区域
- 用户输入框
- 发送按钮

### PreviewPanel
- WebContainer iframe 预览
- 刷新按钮

### ModeToggle
- 切换按钮组
- 当前模式高亮

## 状态管理

使用 Nuxt 4 的 `useState` 进行状态管理：

```typescript
// 状态定义
const currentMode = useState<'chat' | 'code'>('mode', () => 'chat')
const messages = useState<Message[]>('messages', () => [])
```

## WebContainer 集成

使用 `@webcontainer/api` 在浏览器中运行 Node.js 预览服务：

```typescript
// 预览服务启动
// 1. 启动 dev server (如 Vite)
// 2. 通过 iframe 访问 localhost:3000
```

## 注意事项

1. WebContainer 需要 HTTPS 环境或 localhost
2. 当前为模拟阶段，不接入真实 AI
3. 预览仅支持静态 HTML 和简单 Vue 组件
